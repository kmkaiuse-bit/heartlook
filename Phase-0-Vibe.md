# Phase 0: Vibe

## 我想做什麼
建立一個 AI 情感陪伴 Chatbot POC，提供即時的情緒疏導對話，並能識別用戶需求，將高潛在客戶導流至專業諮詢或工作坊。

## 關鍵功能點 (MVP)
•	功能1：AI 擬人化情感交流
利用 System Prompt 設定具備「共情能力」的角色，提供 24/7 的即時傾聽與回應，取代複雜的人工介入。
•	功能2：上下文對話記憶
系統在單次 Session 中能記住前文內容，保持對話流暢度，模仿真人的交流體驗，暫不需複雜的後台學習系統。
•	功能3：關鍵詞觸發導流
當對話出現特定情緒關鍵詞或結束時，生成簡單的情緒摘要，並自動推送諮詢師預約連結或工作坊報名卡片。
